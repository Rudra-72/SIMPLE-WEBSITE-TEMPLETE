Simple Website Template

A clean and minimal multi-section website template built using HTML, CSS, and JavaScript. This template is ideal for beginners and can be used as a starting point for portfolios, company pages, or personal projects.

Features
Fixed navigation bar
Background video in the home section
Multiple sections:
Home
About
Founder
Blogs
Contact
Smooth scrolling navigation
Responsive-ready structure (can be enhanced with CSS)

Project Structure
project-folder/

index.html        # Main HTML file
styles.css        # Styling file
scripts.js        # JavaScript file
background-video.mp4
README.md


Technologies Used
HTML5
CSS3
JavaScript

How to Use

Clone or download this repository:

git clone https://github.com/Rudra-72/SIMPLE-WEBSITE-TEMPLETE.git

Open the project folder
Run the website:
Simply open index.html in your browser
